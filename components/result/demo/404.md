## zh-CN

页面未找到的错误页面。

## en-US

The page you visited does not exist.

```jsx
import { Result, Button } from 'antd';

ReactDOM.render(
  <Result
    status="404"
    title="404"
    subTitle="Sorry, the page you visited does not exist."
    extra={<Button type="primary">Back Home</Button>}
  />,
  mountNode,
);
```
