在元素之间添加横向的间距

## 何时使用

- 需要在组件或元素之间添加横向的间距时使用

## API

### Space

| 参数 | 说明           | 类型                                      | 默认值    | 版本 |
| ---- | -------------- | ----------------------------------------- | --------- | ---- |
| size | 设置空格的大小 | `small` \| `default` \| `large` \| number | `default` |      |
