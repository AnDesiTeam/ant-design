# Alert

- category: Components
- chinese: 警告框

---

显示当前页面在系统层级结构中的位置，并能向上返回。

## 何时使用

- 当系统拥有超过两级以上的层级结构时；
- 当需要向上导航的功能时。

## API

| 参数        | 说明                                       | 类型          | 默认值 |
|----------- |------------------------------------------ | ---------------|--------|
| alertStyle | 警告框的样式，必选 `success/info/warn/error` | string         | 无    |
| message    | 警告框的文字内容                             | string        | 无     |
