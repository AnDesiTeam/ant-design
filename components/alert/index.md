# Alert

- category: Components
- chinese: 通知栏

---

通知栏。

## 何时使用

- 当系统需要向用户显示警告的信息时。

## API

| 参数        | 说明                                                   | 类型        | 默认值 |
|----------- |-----------------------------------------------------   | ---------- |--------|
| alertType  | 通知栏样式，有四种选择`success`、`info`、`warn`、`error` | String      | 无    |
| message    | 通知内容                                              | String      | 无    |
