## en-US

Simply use dom heading elements.

```jsx
const text = "AI for the Common Good";

ReactDOM.render(
  <div>
      <h1>h1. {text}</h1>
      <h2>h2. {text}</h2>
      <h3>h3. {text}</h3>
      <h4>h4. {text}</h4>
      <h5>h5. {text}</h5>
      <h6>h6. {text}</h6>
  </div>,
  mountNode,
);
```
