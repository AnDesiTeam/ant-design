## zh-CN

此布尔属性指定在页面加载时输入应具有焦点。

## en-US

This Boolean attribute specifies that the input should have focus when the page loads.

````jsx
import { Input } from 'antd';

ReactDOM.render(<Input autoFocus placeholder="Auto focused" />, mountNode);
````
