## zh-CN

可以通过设置 `type` 为 `NO_PREMISSION` 选择另一种风格的图片。

## en-US

You can choose another style of `type` by setting image to `NO_PREMISSION`.

```jsx
import { Empty } from 'antd';

ReactDOM.render(<Empty type="NO_PREMISSION" />, mountNode);
```
